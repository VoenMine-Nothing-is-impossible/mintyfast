# mintyfast
Эта удобная и полезная программа для всех, кто много работает за компьютером. Она позволяет получить быстрый доступ к вашим любим программам, встраивая их в шестиячеечную панель быстрого доступа на вашем мониторе. Вы можете убирать и добавлять приложения, которые будут сохранятся не зависимо от работы приложения.
